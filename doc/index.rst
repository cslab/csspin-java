spin_java documentation
=======================

The ``spin_java`` plugin-package for cs.spin serves to provide the necessary
plugins and tools for developing Java-based projects.

The sources can be found at https://code.contact.de/qs/spin/spin_java, while
the released versions are available at
https://packages.contact.de/tools/misc/spin-java.

``spin_java`` requires at least Python 3.9.
